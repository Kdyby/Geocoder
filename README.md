Requirements
------------

Kdyby/Geocoder requires PHP 5.3.2 or higher with cUrl extension enabled.

- [Nette Framework](https://github.com/nette/nette)


Installation
------------

The best way to install Kdyby/Geocoder is using  [Composer](http://getcomposer.org/):

```sh
$ composer require kdyby/geocoder:@dev
```
